# BookNest Chatbot

## Project Description
BookNest Chatbot is an interactive chatbot integrated with an online bookstore platform. It allows users to browse books, get recommendations, and interact with a virtual assistant for a seamless online shopping experience. The project demonstrates full-stack development with a Flask backend, a responsive frontend, and a dashboard for managing conversations.

### Features
- **User-Friendly Chat Interface**: Real-time chatbot for users to ask queries and receive instant responses.
- **Backend Server**: Built with Flask, handling all chat requests and managing book data.
- **Admin Dashboard**: Dashboard for monitoring conversations and managing chat interactions.
- **Book Categories & Recommendations**: Browse books by category and get personalized recommendations.
- **Responsive Design**: Works across desktop and mobile devices.

## Deployment
- The project can be deployed locally using Flask or hosted on platforms like Heroku or Render.  
- To run locally:
  1. Install dependencies: `pip install -r backend/requirements.txt`
  ## Dependencies

This project requires the following Python packages:

- Flask==3.2.1
- Flask-Cors==3.1.2
- Flask-SocketIO==6.4.0
- python-engineio==4.4.0
- python-socketio==6.4.0
- eventlet==0.33.3
  3. Run server: `python app.py`
  4. Open `frontend/chhatbot.html` in your browser
  5. Open `frontend/dashboard.html` in your browser
